〇モデルについて

・ややスケールが小さめですが仕様です。
・ドロワがスカートを貫通してくる場合は「ﾄﾞﾛﾜ貫通避」モーフをご利用ください。

・pmxファイルが4つ入っていますが、長袖/半袖モデルとそれぞれの準標準ボーンモデルです。


Cirno_V11b　	半袖モデル。肩、スカート親、襟にIKを挟んでいる他、P.I.P様の「捩りボーン自動分散プラグイン」を使用しています。
		流し込みで遊ぶ場合はこちらのほうが破綻しづらいです。

Cirno_V11b_LS	長袖モデル。同上。



Cirno_V11b_STD	半袖モデル。腕捩りの自動分散やその他内蔵されているIKを省いて、準標準構造に近づけたモデルです。
		多分各種ボーン改変型プラグインへの耐性があるんじゃないかなと思います、試してないのでなんともですが。

Cirno_V11b_LS	長袖モデル。同上。
