⊂！！使用上の注意点！！⊃

　身長は、135cmを想定して制作しております。そのため、既存モーションの流用等で挙動がおかしくなる場合があります。

　動作が激しいモーションなどでは、スカート等の裏地が突き出る場合があります。

　ニコニコ動画にて、「がに股」ぽいとの指摘を受け、調整を入れましたが、 基本的に、ドロワーズ着用のため、若干 O脚ぎみです。

　先に、quinsiene様(葡萄P様)が、永遠亭の他のメンバーを公開していらっしゃいますが、ボーン構造が異なります。なるべく同じ挙動をするよう調整を行いましたが、細かい部分については個別に修整願います。

　スカートは、スカートAボーン→スカートBボーン→スカート前/横/後ボーンの順で、おおまかな調整〜細かな調整という考えの元にボーン入れを行っています。そのため、脚が大きく開くようなポーズではボーンだけでは破綻を吸収しきれない場合があります。その場合は、後述する「スカート拡大」「スカート上部拡大」モーフを併用してください。

　初音ミク等のモーション流用時は、「腕ＩＫ」を OFFにしてください。

⊂各種IKとコントロールリグ⊃


以下の図のようになっております。

　スカート裾部分については、標準モデルと同名称にしています。従って、標準モデルのスカートが揺れる場合、本モデルでは裾の一部が動きます。
